Each file contains the captured packets and corresponds to each required configuration.

pkt-capture_sem-perdas.pcap => No forced losses
pkt-capture_perdas-1.pcap   => 1% of loss
pkt-capture_perdas-10.pcap  => 10% of loss
pkt-capture_perdas-20.pcap  => 20% of loss

The "analysis" folder contains the document files required to correctly explain the results obtained.
